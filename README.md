# ThinkSpace

ThinkSpace is a full-stack note-taking web application that allows users to create, edit, and manage notes through a clean and responsive interface.

## Features
- Create, edit, and delete notes (CRUD functionality)
- Responsive UI built with React and Tailwind CSS (DaisyUI)
- Real-time data updates with RESTful API integration
- Loading states and error handling
- Rate limiting handling (HTTP 429 support)

## Tech Stack
- Frontend: React.js, JavaScript, Tailwind CSS, DaisyUI
- Backend: Node.js, Express.js
- Database: MongoDB
- API Communication: Axios
