# ThinkSpace
ThinkSpace is a full-stack note-taking web application that allows users to create, edit, and manage notes through a clean and responsive interface.

## Project Overview
This project demonstrates full-stack development through a notes management system with CRUD functionality, client-side routing, API integration, and production deployment. It was built to demonstrate practical skills in modern web development, including frontend UI design, backend API development, and database integration.

## Features
- Create, edit, update, and delete notes
- Responsive user interface built with React and Tailwind CSS
- RESTful API integration using Axios
- Loading states and error handling
- Rate limiting support for API requests (HTTP 429 support)
- Full-stack deployment with frontend and backend combined

## Tech Stack
- Frontend: React.js, JavaScript, Tailwind CSS, DaisyUI
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- API Communication: Axios
- Deployment: Render

## Deployment
The application is deployed on Render using a combined frontend and backend setup.
